🔑 Générer les clés VAPID

Ces clés sécurisent les notifications push. Génère-les une seule fois et note-les.

⚠️ Ce générateur utilise ECDSA P-256 — le seul format compatible avec web-push.
PUBLIC KEY — dans .env ET dans Supabase secrets
PRIVATE KEY — UNIQUEMENT dans Supabase secrets (ne partage jamais)

Où les coller ?

1. Dans .env à la racine du projet :

VITE_VAPID_PUBLIC_KEY="…"

Remplace la valeur existante si elle existe déjà.
2. Dans Supabase Dashboard
→ Project Settings → Edge Functions → Secrets

Ajoute (ou remplace) ces 3 secrets :

VAPID_PUBLIC_KEY = la public key

VAPID_PRIVATE_KEY = la private key

VAPID_EMAIL = mailto:contact@ascension20.fr
3. Dans Supabase SQL Editor
→ New query → colle et exécute le contenu de :

supabase/migrations/20260527210000_push_subscriptions.sql
4. Déploie l'edge function via Lovable
Utilise le prompt fourni dans la conversation pour déployer send-notifications.
